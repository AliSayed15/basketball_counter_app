🏀 Basketball Counter App

A simple yet powerful basketball score tracking application built with Flutter, designed to provide real-time score updates with a clean and intuitive user interface.

📌 Overview


The Basketball Counter App is a mobile application that allows users to track scores for two teams during a basketball game. It focuses on simplicity, performance, and usability, making it ideal for quick matches, training sessions, or casual gameplay.


This project demonstrates practical usage of Flutter UI design and state management using Cubit (BLoC).


✨ Features
➕ Add points (1, 2, 3) for each team
🔄 Reset scores instantl
⚡ Real-time UI updates using state management
🎯 Clean and responsive user interface
🧠 Built with Cubit for efficient state handling
🛠️ Tech Stack
Flutter
Dart
Cubit (BLoC)
📂 Project Structure
lib/
├── cubit/        # State management logic
├── views/        # UI screens
├── widgets/      # Reusable components
└── main.dart     # App entry point
🚀 Getting Started
Prerequisites
Flutter SDK installed
Android Studio / VS Code
Emulator or real device
Installation
git clone https://github.com/AliSayed15/basketball_counter_app.git
cd basketball_counter_app
flutter pub get
flutter run
🎮 How to Use
Open the app
Choose a team (Team A / Team B)
Add points using the buttons (+1, +2, +3)
Reset scores anytime using the reset button
